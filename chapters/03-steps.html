Шаг 4

...так прошёл месяц, другой, третий и вот Вы вернулись с готовым приложением. Поздравляем!

Теперь необходимо его установить.

Мы не рекомендуем использовать площадку серверов ЕГУ И.А.Бунина.

В период с ноября 2011 по май 2012 был написан проект "Виртуальный университет", включающий в себя площадку "Портфолио достижений" и Систему курсов для портфолио". В ноябре 2012 приказом Зайцева А.А. он был разделен на части. Оба проекта накрылись.

Наиболее предпочтительным является размещение Вашего сайта на хостинге с положительной репутацией, находящимся на территории РФ.

Для аспирантов ЕГУ мы не рекомендуем использовать сервер другой образовательной организации.

В связи с невозможностью установки на серверах ЕГУ, проект "Траектория обучения" был установлен на сервере университета Н.Новгорода. В результате автор проекта был обвинен Белкановым Н.А. во взаимодействии с другими вузами. Проект был отклонён.

Хотя, если у Вас googleAps приложение, то выбирать не приходиться.

Шаг 5

Рассмотрим выходные данные обработчика событий

Для актора и объекта вывод будет представлен в следующем виде:

var stmt = {
"actor": actor, "verb": ADL.verbs.experienced, "context": {
"contextActivities": {
"parent": [
{
"id": moduleID,
"definition": {
"name": {
"en-US": "xAPI for jQuery Demo"
}, "description": {
"en-US": "A sample HTML5 mobile app with xAPI tracking."
}
},
"objectType": "Activity"
}
]
}
},
"object": {
"id" : activityID, "objectType": "Activity", "definition": {
"name": { "en-US": "xAPI jQuery Mobile " + chapter + " " + pagename }
}
}
};

Вывод состояния какого-либо модуля может быть осуществлен через функцию baseActivity

var baseActivity = {
"id": moduleID,
"definition": {
"name": { "en-US": moduleName }, "description": { "en-US": "...описание процесса" }
},
"objectType": "Activity"
};
```

Отправка данных о состоянии производится функцией содержащей XAPIWrapper

function courseLaunched() {
doConfig(); // statement for launching content var stmt = {
"actor": actor, "verb": ADL.verbs.launched, "object": baseActivity
}; // Send launched statement ADL.XAPIWrapper.sendStatement(stmt);
}
